Introduction

嗨！我是亭予！

【擁有上百間網站規劃經驗，捉住程式盲點和解決 BUG】

進入公司後，我發現我有著能夠捉住程式盲點或是解決 BUG 的能力。透過大量蒐集網站的 UI/UX 資料，我能夠與工程師們有效地溝通，共同改善程式，這也使我成為處理客製網站案件的首選人。

【探索前端世界】

隨時間深入瞭解 Web 產業，我也漸漸領略到前端工程師這個領域的魅力。

我的職位是專案規劃，並不需要直接接觸代碼。然而，有一次我偶然間看到設計師如何透過簡單的代碼調整來改變網站的外觀。這激發了我強烈的好奇心，我的設計師同事簡單地教導了我如何使用開發者工具微調 CSS。從那一刻起，我踏上了自學前端的不歸路（笑）。

【自主學習的冒險】

從最初的 HTML 開始扎根，也學到「語意化」這個重要觀念，投入了大量的時間和精力，學習 HTML、CSS 和 JavaScript 等基礎知識，不斷追求更深入的學習，研究各種前端框架和工具。不僅專注於程式邏輯，我也追求網站的美感和流暢度，不斷挑戰自己，嘗試融入新的設計風格和動態效果。

【進一步的學習和成長】

在 2022 年，我參加了李建杭（Amos）老師的「暴力網頁新手入門班」以及「 RWD 網頁原來是醬子」線上課程。在這些課程中，我深刻理解了如何手刻響應式網頁，寫出簡潔的代碼。這對我的影響深遠，使我在後續的程式撰寫中常思考如何更加精簡，如何使函數更具再利用性。

針對 JavaScript 及 React.js 的部分購買了 Udemy 課程，持續進修中，也透過閱讀資料，不斷地精進自己的技術。
