Introduction

嗨！我是亭予！

在成為前端工程師之前，我曾是專案規劃，擁有上百間網站規劃經驗，能夠捉住程式盲點和解決 BUG，並通過大量蒐集 UI/UX 資料，與工程師有效溝通，共同改善程式。

一次偶然機會下，設計師教我通過簡單的代碼改變網站外觀，這激發了我對前端技術的興趣，從此踏上了學習前端的旅程。

在跳跳，我開始了第一份前端工程師工作。我從零開始學習 Canvas 和 Three.js 等視覺技術，並將這些技術應用於視覺創作及製作網頁遊戲，因為遊戲製作，讓我也順理成章接觸到物理引擎、效能優化以及多人連線等相關技術，這些經驗大大提升了我的前端技術能力、創意設計能力和問題解決能力。

在個人練習專案中我也用 React 開發了一個購物車系統和一個 Threads 社群平台實作練習。這些專案讓我深入了解了如何將商業需求轉化為技術實現，同時也提升了我在 React 和前端開發方面的技能和實踐經驗。

在未來的工作中，也希望能有更多與團隊協作的機會，通過團隊之間的互動和學習，進一步提升自己的專業技能。
